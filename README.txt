خطوات الرفع:
1. أنشئ حساب GitHub وارفع الملفات.
2. ادخل إلى Railway.app وسجل دخولك بحساب GitHub.
3. اربط المشروع الجديد مع GitHub واختر هذا المشروع.
4. اضغط على Settings > Variables وأضف متغير TOKEN مع توكن البوت الخاص بك.
5. شغل البوت واستمتع!

ملاحظة:
- تأكد أن التوكن صالح ويبدأ بـ: 123456789:AA...
- لا تشارك التوكن مع أحد.
- البوت يبقى يعمل 24/24 مجانًا.
